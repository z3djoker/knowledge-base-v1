# AI Knowledge Base

`knowledge-base-v1` is a Next.js foundation for an AI knowledge base product.
It includes the application shell, primary routes, TypeScript, Tailwind CSS,
and static placeholder screens. AI, document ingestion, persistence, and
authentication are intentionally not implemented yet.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- ESLint
- npm

## Architecture

```text
src/
  app/
    admin/page.tsx    Admin foundation page
    chat/page.tsx     Chat foundation page
    upload/page.tsx   File upload foundation page
    globals.css       Tailwind import and global theme tokens
    layout.tsx        Root metadata, fonts, and document shell
    page.tsx          Modern homepage
  components/
    site-shell.tsx    Shared page chrome and navigation
  lib/
    navigation.ts     Route definitions used by the shell
public/
  Static assets generated by create-next-app
```

The project uses the App Router and keeps shared UI in `src/components`.
Route-specific screens live under `src/app`, and shared configuration data lives
under `src/lib`.

## Routes

- `/` - Homepage with project status and module cards
- `/upload` - File upload placeholder page
- `/chat` - Chat placeholder page
- `/admin` - Admin placeholder page

## Development

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Run linting:

```bash
npm run lint
```

## Current Scope

Implemented:

- Project foundation
- Shared app shell
- Homepage
- File upload, chat, and admin pages
- README architecture notes

Not implemented yet:

- AI model integration
- Retrieval augmented generation
- Document parsing, chunking, or indexing
- File storage
- Database
- Authentication and authorization
